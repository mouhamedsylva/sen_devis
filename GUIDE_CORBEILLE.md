# 🗑️ Guide d'Utilisation de la Corbeille

## 📖 Introduction

La corbeille de SenDevis vous permet de gérer les éléments supprimés (devis et produits) en toute sécurité. Vous pouvez restaurer des éléments supprimés par erreur ou les supprimer définitivement.

## 🎯 Accéder à la Corbeille

### Depuis les Paramètres
1. Ouvrez l'application SenDevis
2. Allez dans **Paramètres** (icône ⚙️ en bas à droite)
3. Dans la section **Gestion des données**, cliquez sur **Corbeille**
4. Un badge rouge indique le nombre d'éléments dans la corbeille

## 📱 Interface de la Corbeille

### Onglets de Filtrage
La corbeille dispose de 3 onglets pour filtrer les éléments :
- **Tout** : Affiche tous les éléments supprimés
- **Devis** : Affiche uniquement les devis supprimés
- **Produits** : Affiche uniquement les produits supprimés

### Informations Affichées
Pour chaque élément, vous verrez :
- 📄 **Icône** : Bleue pour les devis, orange pour les produits
- **Titre** : Nom du produit ou numéro du devis
- **Date** : Date de suppression
- **Actions** : Boutons Restaurer et Supprimer

## ♻️ Restaurer un Élément

### Étapes
1. Trouvez l'élément que vous souhaitez restaurer
2. Cliquez sur le bouton **Restaurer** (icône ↻ verte)
3. Confirmez la restauration dans le dialogue
4. L'élément est restauré et réapparaît dans sa liste d'origine

### Résultat
- ✅ L'élément est retiré de la corbeille
- ✅ L'élément réapparaît dans la liste des devis ou produits
- ✅ Un message de confirmation s'affiche

## 🗑️ Supprimer Définitivement un Élément

### ⚠️ Attention
La suppression définitive est **irréversible**. L'élément sera supprimé de la base de données et ne pourra plus être récupéré.

### Étapes
1. Trouvez l'élément que vous souhaitez supprimer définitivement
2. Cliquez sur le bouton **Supprimer** (icône 🗑️ rouge)
3. Lisez attentivement le message d'avertissement
4. Confirmez la suppression définitive

### Résultat
- ❌ L'élément est supprimé définitivement de la base de données
- ❌ L'élément disparaît de la corbeille
- ⚠️ Cette action est irréversible

## 🧹 Vider la Corbeille

### ⚠️ Attention
Vider la corbeille supprime **tous les éléments** définitivement. Cette action est **irréversible**.

### Étapes
1. Cliquez sur le menu **⋮** (trois points) en haut à droite
2. Sélectionnez **Vider la corbeille**
3. Lisez attentivement le message d'avertissement
4. Confirmez l'action

### Résultat
- ❌ Tous les éléments sont supprimés définitivement
- ❌ La corbeille est vide
- ⚠️ Cette action est irréversible

## 🔄 Actualiser la Corbeille

Pour actualiser la liste des éléments :
- Tirez vers le bas (pull-to-refresh) sur la liste
- Ou quittez et revenez dans la corbeille

## 💡 Conseils d'Utilisation

### Bonnes Pratiques
1. **Vérifiez avant de supprimer** : Assurez-vous que vous ne voulez vraiment plus l'élément
2. **Restaurez rapidement** : Si vous supprimez par erreur, restaurez immédiatement
3. **Nettoyez régulièrement** : Videz la corbeille de temps en temps pour libérer de l'espace
4. **Utilisez les onglets** : Filtrez par type pour trouver rapidement un élément

### Cas d'Usage
- **Suppression accidentelle** : Restaurez l'élément depuis la corbeille
- **Nettoyage de printemps** : Supprimez définitivement les vieux éléments
- **Libération d'espace** : Videz la corbeille pour optimiser la base de données

## ❓ Questions Fréquentes

### Combien de temps les éléments restent-ils dans la corbeille ?
Les éléments restent dans la corbeille indéfiniment jusqu'à ce que vous les supprimiez définitivement ou vidiez la corbeille.

### Puis-je restaurer plusieurs éléments à la fois ?
Actuellement, vous devez restaurer les éléments un par un. Une fonctionnalité de sélection multiple pourrait être ajoutée dans une future version.

### Que se passe-t-il si je supprime un devis lié à un client ?
Le devis est supprimé mais le client reste intact. Vous pouvez restaurer le devis à tout moment.

### Puis-je voir les détails d'un élément dans la corbeille ?
Actuellement, vous voyez uniquement le titre et la date de suppression. Pour voir les détails, vous devez d'abord restaurer l'élément.

### La corbeille fonctionne-t-elle hors ligne ?
Oui, la corbeille fonctionne entièrement hors ligne comme le reste de l'application.
