## 🔄 Changer le Logo

Pour remplacer le logo à l'avenir :

1. Remplace `assets/icons/logo.png` par ton nouveau logo
2. Exécute `flutter clean && flutter pub get`
3. Relance l'application

**Recommandations :**
- Format : PNG avec transparence
- Dimensions : Minimum 512x512 pixels
- Taille : < 500 KB
